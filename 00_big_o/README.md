# Big-O Notation

Big-O notation is used to describe the **time and space complexity** of algorithms.
It helps evaluate how an algorithm scales as the input size grows.

Understanding Big-O is essential for:
- Writing efficient code
- Comparing algorithms
- Passing technical interviews
