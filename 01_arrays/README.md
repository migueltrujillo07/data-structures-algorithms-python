# Arrays (Lists) in Python

Arrays are one of the most fundamental data structures.
In Python, arrays are implemented using **lists**.

They store elements in contiguous memory locations and allow fast access by index.

---

## Common Operations

| Operation | Time Complexity |
|---------|----------------|
| Access | O(1) |
| Update | O(1) |
| Search | O(n) |
| Insert | O(n) |
| Delete | O(n) |

---

## When to Use Arrays
- Sequential data
- Fast access by index
- Sliding window problems
- Two pointers technique
